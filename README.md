# Testes de Hipóteses

## Os testes realizados foram:

### Testes para Normalidade:
- **Kolmogorov-Smirnov**
- **Shapiro-Wilk**
- **Shapiro-Francia**

### Testes para Homogeneidade de Variâncias:
- **Bartlett**
- **Cochran**
- **Levene**

### Testes para Médias Populacionais:
- **Teste Z**
- **Teste T-Student**
